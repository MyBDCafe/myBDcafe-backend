# :stars:내.생.카.(내 최애의 생일 카페)
> 내생카는 아이돌 팬덤이나 서브컬쳐 장르에서<br>
> 자신이 좋아하는 존재(최애)의 기념일에 열리는<br>
> 이벤트 카페들의 일정을 보기 편하게 저장하고, 공유하는 어플리케이션입니다<br><br>
> 개발기간 2024.01~(진행중)
<br>
